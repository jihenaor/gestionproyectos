# Gestion Proyectos

Sistema de gestión de proyectos para Comfamiliar Risaralda.

## Estructura

- `/backend` - Backend Spring Boot
- `/frontend` - Frontend Angular
- `/infrastructure` - Configuración de infraestructura
- `/scripts` - Scripts de despliegue

## Tecnologías

- Backend: Spring Boot 3.4, Java 21, JPA/Hibernate, OAuth2 JWT
- Frontend: Angular 19
- Base de datos: IBM i (AS400) / H2 (desarrollo)
- Contenedores: Docker
